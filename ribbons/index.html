CSS Ribbons

Apparently, ribbons are 100% possible using only CSS.

The Markup

<div id="wrapper">

<div class="ribbons">

	<span class="triangle-left"></span>
		<h1>CSS Ribbons</h1>
	<span class="triangle-right"></span>
	
</div>
</div>

How does it all work?

It might look like there is a lot going on here. When you break it down, it's really not so bad. The triangles are quite simple to accomplish. When you put a border on an element, the browser renders that border with a 45 degree angle at both ends. Try this out:

HTML

<span class="triangle-left">

CSS

.triangle-left {
	width: 0; heigh: 0;
	border-color: transparent #000 transparent transparent;
	border-style: solid;
	border-width: 25px;
}

The Result

What's happening here is you are telling the browser to render 3 transparent borders and one black. In this case all you are seeing is the right hand border. If you change the colors from transparent, you'll find some interesting results.
